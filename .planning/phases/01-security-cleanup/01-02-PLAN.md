**Part B — Remove hardcoded AWS credentials from `backend/fetch_latest_news.py`.**

Read the full current `backend/fetch_latest_news.py`. It contains:
```python
AWS_ACCESS_KEY_ID = 'AKIA3FLD2AYWN7TZUV5X'

Replace those three lines with environment variable reads using `os.environ.get`:

```

Add `import os` at the top of the file (after `import boto3` — the file currently imports yfinance, json, config, datetime, boto3). The `boto3.client()` call on line 40 uses these variables and does not need to change.

Preserve all other logic in the file exactly — only the three credential assignment lines change (plus adding `import os`).
